# 🎬 Snaptube-like Video Downloader

**Super simple video downloader - just like Snaptube!**

## 🚀 How to Use

### Method 1: Simple Command
```bash
./download_video.sh "https://www.youtube.com/watch?v=VIDEO_ID"
```

### Method 2: Direct Python
```bash
cd backend
source venv/bin/activate
cd ..
python3 snaptube.py "https://www.youtube.com/watch?v=VIDEO_ID"
```

### Method 3: Download Audio (MP3)
```bash
./download_video.sh "https://www.youtube.com/watch?v=VIDEO_ID" --audio
```

## 📱 What It Does

1. **Paste any video URL** (YouTube, Instagram, TikTok, etc.)
2. **Automatically downloads** to your Downloads folder
3. **Works like Snaptube** - no complex setup needed!

## 🎯 Examples

```bash
# Download YouTube video
./download_video.sh "https://www.youtube.com/watch?v=dQw4w9WgXcQ"

# Download Instagram video
./download_video.sh "https://www.instagram.com/p/VIDEO_ID/"

# Download TikTok video
./download_video.sh "https://www.tiktok.com/@user/video/VIDEO_ID"

# Download as MP3 audio
./download_video.sh "https://www.youtube.com/watch?v=dQw4w9WgXcQ" --audio
```

## 📁 Where Files Are Saved

All downloads go to: `~/Downloads/` (your Downloads folder)
